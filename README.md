Info to be able to run the app: 
1. Install Dependencies
Before running the app, ensure you have the necessary packages installed. Open your terminal and run: pip install -r requirements.txt 
Note: If you have already installed these requirements, you can skip this step.
2. Run the Application
Open the file named run.py in your editor.
Click the 'Run' icon at the top of the screen (the small play button/triangle).
3. Access the Website
Once the script is running, look at the output in your terminal. You will see a URL ending in :5000 (e.g., http://127.0.0.1:5000).
Hold the Ctrl key and click the link to open the website automatically in your browser.

Additional information when starting the application: When you arrive at the login screen, you will see below the password bar what information you need to enter to log in (email/password). When you log in, as a depot employee, you will not see certain things that you would see if you logged in as a finance employee, and vice versa. If you log in as an admin, you will see everything.
